# Super Simple Sudoku Version 3

A Vue.js 3 application for playing Sudoku with multiple difficulty levels.

## Prerequisites

- Node.js (version 14 or higher recommended)
- npm (comes with Node.js)

## Installation

1. Navigate to the app directory:
```bash
cd app
```

2. Install dependencies:
```bash
npm install
```

## Running the Application

To start the development server:
```bash
npm run serve
```

The application will be available at `http://localhost:8080` (or another port if 8080 is in use).

## Building for Production

To create a production build:
```bash
npm run build
```

The built files will be in the `dist` directory.

## Features

- **Multiple Difficulty Levels**: Easy, Medium, Hard, and Super Hard
- **Hint System**: Get hints for empty cells with visual animation
- **Auto-Solve**: Solve the puzzle automatically
- **3D Visual Effects**: Filled numbers have a 3D teal appearance
- **Interactive Gameplay**: Click cells to enter numbers
